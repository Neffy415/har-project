DETAILED PIPELINE BREAKDOWN:

Step 1: Extract Features (build_dataset.py) - 15-40 minutes
   ├─ Load YOLO v8 Nano model
   ├─ Load MobileNetV2 pre-trained model
   ├─ For each of 12 videos:
   │  ├─ Read video frames
   │  ├─ Detect people with YOLO
   │  ├─ Crop person region
   │  ├─ Extract 1280-D features with MobileNetV2
   │  └─ Accumulate into 30-frame sequences
   └─ Save X.npy (sequences) and y.npy (labels)

Step 2: Train Model (train_data.py) - 5-15 minutes
   ├─ Load X.npy and y.npy
   ├─ Build LSTM neural network
   │  ├─ Layer 1: LSTM(128) with return_sequences=True
   │  ├─ Dropout(0.3)
   │  ├─ Layer 2: LSTM(64)
   │  ├─ Dropout(0.3)
   │  ├─ Dense(64) with ReLU
   │  └─ Dense(4) with Softmax (4 activity classes)
   ├─ Compile with Adam optimizer
   ├─ Train for 10 epochs on extracted features
   └─ Save model.h5

Step 3: Real-time Inference (main.py) - Runs indefinitely
   ├─ Load trained model.h5
   ├─ Load YOLO and MobileNetV2 models
   ├─ Open webcam (VideoCapture(0))
   ├─ For each frame:
   │  ├─ Detect people with YOLO
   │  ├─ Extract features with MobileNetV2
   │  ├─ Buffer last 30 frames
   │  ├─ Predict activity class
   │  ├─ Display prediction with confidence
   │  └─ Show bounding box
   └─ Press 'Q' to exit
